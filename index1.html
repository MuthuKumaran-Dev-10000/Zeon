Introduction to Zeon 🌟

Zeon is a high-level programming language designed for easy coding, interoperability, and rapid development. It combines the simplicity of scripting languages with the power of modern programming paradigms. Zeon supports interoperability with languages like Java and Python, making it a versatile choice for developers. Additionally, Zeon integrates seamlessly with HTML for mobile app development, enabling you to create stunning user interfaces with ease. 🎨

Syntax Overview 🖋️

Zeon's syntax is clean, intuitive, and easy to learn. Here's a quick overview of the key syntax elements:

let x : 10; // Declares a variable x with value 10
print("Hello, Zeon!"); // Prints to the console
if (x > 10) {
    print("x is greater than 10");
}

Variables and Data Types 🧱

Zeon supports the following data types:

  • Numbers: let age : 25;
  • Strings: let name : "Zeon";
  • Booleans: let isActive : true;
  • Lists: let numbers : [1, 2, 3];
  • Maps: let person : {"name": "Zeon", "age": 25};

Control Structures 🎛️

Zeon provides familiar control structures for decision-making and looping:

if (x > 10) {
    print("x is greater than 10");
} elseif (x == 10) {
    print("x is equal to 10");
} else {
    print("x is less than 10");
}

Functions 🛠️

Functions in Zeon are declared using the fn keyword:

fn greet(name) {
    print("Hello, ", name, "!_n");
}

Escape Sequences 🎨

Zeon's escape sequences make it easy to format and style text output:

print("_bold _underline Welcome to Zeon!_n");
print("_green This is green text._n");

Interoperability 🤝

Zeon is designed to work seamlessly with other programming languages like Java and Python:

java {
    System.out.println("Hello from Java!");
}

HTML Integration for Mobile App Development 📱

Zeon makes it easy to develop mobile apps by integrating with HTML:

html {
    <!DOCTYPE html>

    <head>
        <title>Zeon Mobile App</title>
    </head>

Examples 🚀

Here are some examples to get you started:

print("_blue Hello, _bold World!_n");

fn fibonacci(n) {
    let a : 0;
    let b : 1;
    while (n > 0) {
        print(a, " ");
        let temp : a;
        let a : b;
        let b : temp + b;
        let n : n - 1;
    }
}
